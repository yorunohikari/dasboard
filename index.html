▦ DasBoard

Hey, I made (vibecoded actually) this webapp because I just hate Figma. JK lol, I just like making my own tools. Funny story, I made this for my algorithm learning for speedcubing.

This webapp is fully local — no tracker, no privacy trespassing whatsoever. I'm not Zuck, nor Palantir. All data lives in localStorage, hence the limited storage.

Beware of data loss if you flush the cache or whatever. You can backup your data by manually exporting and importing using the buttons in Settings.

Okay, have fun. ✌

Coming soon — PWA support.
Install DasBoard to your homescreen. Works fully offline after first load. No app store, no install wizard — just a browser.

  • Boards & cards with drag-and-drop
  • Infinite canvas — pan & zoom
  • Double-click to edit cards & boards
  • Image upload with WebP compression
  • Masonry columns per board (1–8)
  • Custom board colors, light & dark mode
  • Context menu — cut, copy, paste, duplicate
  • Move to board, move to top/bottom

  • Export & import as JSON
  • Storage meter, font size settings
  • No account. No server. No tracking.
